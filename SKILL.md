# 资产配置建议 Skill（知行温度计）

基于有知有行温度计数据，为用户提供 A 股市场估值分析和资产配置建议。

## Trigger Conditions

当用户想要：
- 了解当前 A 股市场温度/估值水平
- 获取温度计数据（全市场温度、指数温度、宏观数据）
- 判断现在是否适合买入/加仓/减仓
- 制定定投策略或资产配置方案
- 查看主要指数的估值和收益率

**关键词**：
- "温度计"、"市场温度"、"全市场温度"
- "现在能不能买"、"现在该不该加仓"
- "低估"、"高估"、"中估"
- "资产配置"、"定投建议"
- "指数估值"、"PE"、"PB"
- "有知有行"、"知行温度计"

## Workflow

### 1. 获取温度计数据

使用 `mcp__web_reader__webReader` 工具获取最新温度计数据：

```
URL: https://youzhiyouxing.cn/thermometer?format=md
```

> **注意**：该页面为客户端渲染 SPA，`curl` / `requests` 只能获取空壳 HTML。
> 必须使用 `webReader`（浏览器内核渲染）才能获取完整 Markdown 内容。

解析返回的 Markdown 内容，提取以下数据：

- **全市场温度**：0°-100° 的温度值、温度带（低估/中估/高估）、温度趋势（上升/下降）
- **更新时间**：数据最后更新日期
- **指数观察**：12 个主要指数的温度、内在收益率、股息率
- **宏观数据**：债市温度、10 年期国债收益率、GDP 增速、CPI 增速

### 2. 数据解析

从 Markdown 中提取结构化数据，重点关注：

| 数据项 | 说明 |
|--------|------|
| 全市场温度 | 判断整体市场冷热 |
| 温度趋势 | 上升/下降，判断市场方向 |
| 各指数温度 | 判断各板块估值差异 |
| 内在收益率 | 评估指数投资价值 |
| 股息率 | 评估分红收益 |
| 债市温度 | 判断股债性价比 |

### 3. 生成投资建议

根据温度带和具体数据，按照以下框架生成建议。

#### 温度带划分

| 温度带 | 范围 | 历史发生概率 | 买入持有 5 年盈利概率 | 操作建议 |
|--------|------|-------------|---------------------|----------|
| 低估 | 0°-30° | 40% | >95% | 积极买入，加大定投 |
| 中估 | 30°-70° | 38% | >90% | 正常定投，保持仓位 |
| 高估 | 70°-100° | 22% | >35% | 减少买入，逐步兑现 |

#### 建议生成规则

**低估区（0°-30°）**：
- 整体建议：加大投资力度，这是最佳买入时机
- 定投策略：加倍定投金额（1.5-2 倍）
- 持有建议：长期持有（5 年以上），盈利概率 >95%
- 重点关注：低温度指数

**中估区（30°-70°）**：
- 整体建议：维持正常投资节奏
- 定投策略：按计划正常定投
- 持有建议：中长期持有（3-5 年），盈利概率 >90%
- 重点关注：寻找低温度的细分指数机会

**高估区（70°-100°）**：
- 整体建议：减少新投入，考虑逐步兑现收益
- 定投策略：暂停或减少定投金额
- 持有建议：降低仓位预期，注意风险控制
- 重点关注：盈利概率仅 >35%，需谨慎

#### 股债配置参考

根据全市场温度和债市温度的对比：
- 股市温度 < 债市温度 → 偏股配置更优
- 股市温度 > 债市温度 → 需权衡风险，考虑增配债券
- 股市低估 + 债市高估 → 极佳的股债切换机会

#### 指数轮动建议

根据各指数温度的差异，提供板块配置建议：
- 优先关注低估指数（温度 < 30°）
- 适度配置中估指数（温度 30°-50°）
- 谨慎对待高估指数（温度 > 70°）

## 数据源说明

- **数据来源**：有知有行 (youzhiyouxing.cn)
- **API 格式**：`https://youzhiyouxing.cn/thermometer?format=md`（Markdown 格式，适合 LLM 解析）
- **更新频率**：每个交易日收盘后更新（通常为交易日 20:00）
- **历史数据**：`https://youzhiyouxing.cn/data/market?format=md`
- **宏观数据**：`https://youzhiyouxing.cn/data/macro?format=md`

### 备选方案

如果 `webReader` 不可用：
1. 使用 `WebSearch` 搜索"有知有行 温度计"获取最新信息
2. 使用 `mcp__qieman__GetLatestQuotations` 获取市场行情作为补充

## 重要声明

1. 温度计数据仅供参考，不构成任何投资建议或承诺。
2. 历史数据不代表未来表现，过去盈利概率不保证未来收益。
3. 投资有风险，入市需谨慎，请结合自身风险承受能力和投资期限综合判断。
4. 温度计基于全市场等权估值（PE + PB），反映整体估值水平，不适用于个股或行业判断。
5. 数据来源为有知有行，本 Skill 不对数据准确性和完整性做任何担保。
6. 建议投资期限 3-5 年以上，短期市场波动可能导致浮亏。
